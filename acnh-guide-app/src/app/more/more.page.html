À propos

ACNH Simple Guide est une application Open Source, dont le code complet est disponible sur GitHub à l'adresse suivante : https://github.com/souyahia/acnh-guide.

Les données de cette application ont été récupérées depuis https://www.animalcrossing-online.com/ à l'aide d'un webscraper développé en NodeJs (Request / Cheerio). L'application en elle même est basée sur Ionic 5 et Capacitor.
